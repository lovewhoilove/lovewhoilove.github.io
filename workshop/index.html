Openlayers 研讨会

英文原文链接：https://openlayers.org/workshop/en/ 。

引言

欢迎来到Openlayers研讨会，本研讨会旨在让你全面了解OpenLayers作为Web地图解决方案的总体情况。

环境设置

关于这些说明，假设你是从最新的研讨会版本的openLayer-works-en.zip归档文件开始。此外，你还需要安装Node v16或更高版本，使得研讨会可以运行在开发服务器上。
解压压缩包后，进入openlayers-workshop-en目录，并安装一些额外的依赖项:

现在，你已经准备好启动开发服务器了，除了为OpenLayers库提供模块捆绑器之外，它还提供了研讨会文档资料。

这将启动一个开发服务器，你可以在其中阅读研讨会文档并完成练习。你应该能够通过在http://localhost:5173/上看到弹窗来确认已正常运行项目,并且可以在http://localhost:5173/doc/上阅读研讨会文档。

概述

本研讨会以一组模块的形式呈现，在每个模块中，你将执行旨在实现该模块特定目标的任务，每个模块都建立在以前模块中吸取的教训之上，旨在迭代地建立你的知识库。
本研讨会将介绍以下模块：

  • 基础概念：了解如何在网页中添加地图。
  • 矢量数据：处理矢量数据
  • 移动地图和传感器：带有GPS和指南针的移动地图
  • GeoTIFF渲染：从GeoTIFF数据源生成数据切片并将其可视化
  • 矢量瓦片和Mapbox样式：使用矢量切片创建漂亮的地图
  • WebGL点的渲染：使用WebGL来渲染点
  • 部署：构建生产应用程序

基础知识

确保你已经完成了安装依赖项的安装说明，并使开发服务器运行起来。
既然我们已经配置了开发环境，让我们开始使用OpenLayers地图创建一个简单的Web页面，并理解代码。
在OpenLayers中，地图是渲染到网页上的图层的集合。要创建地图，你需要一些用于创建地图视图的标记元素(如<div>元素)、一些使地图视图在页面上具有适当的尺寸的样式，以及地图初始化代码。
OpenLayers支持不同类型的图层：

  • 切片图层，平铺的栅格数据切片集合(原文：Tile layers for tiled raster tile sets，真拗口)
  • 影像图层，影像文件或根据地图范围需要提供的影像(原文：Image layers for static images or images that are provided on demand for the map’s extent)
  • 矢量图层，矢量数据文件或地图当前范围的矢量数据(原文：Vector layers for vector data from static files or for the map’s current extent)
  • 矢量切片图层，平铺的矢量切片集合(原文：Vector tile layers for tiled vector tile sets)

除了图层和视图之外，地图还可以配置一些控件(contorl，即地图上方的UI元素)和交互(interaction，即对地图上的触摸或(鼠标)指针手势做出反应的组件)。
