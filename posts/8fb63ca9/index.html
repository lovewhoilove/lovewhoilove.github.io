前言

注：下图中控制台给出的黄色警告信息，这个目前没有找到解决方法，后续得看官方如何优化了。

另外，我们认真翻看 VS Code 命令行的打包日志，还能发现警告信息，具体如下图所示：提示打包出来的文件超过了推荐的大小上限 244 KiB。

其实上述问题可以忽略，只能说 Cesium 官方针对 npm 方式引入方式优化的不够好了。后来，我又咨询了一个大佬，这个可能影响项目线上发布后首次加载的速度，尤其是想服务器带宽小的，就比较明显。

针对这种情况，我尝试了 Cesium本地环境配置 一文中介绍的配置方法，但在打包后仍然会出现上述警告。代码详见：下载源代码 ，项目名称为：vue3-cesium-local。
